Technisch:

  • Textvorbereitung:
    • Der Text wurde gereinigt [^a-zA-Z.:;—?!] und in Zeilen aufgeteilt.
    • Ein Index-zu-Wort-Wörterbuch wurde mithilfe eines Tokenizers generiert.
  • Sequenzierung:
    • Die Zeilen wurden in N-Grammen Sequenzen von Ganzzahlen umgewandelt.
    • Alle Sequenzen wurden von links auf die Länge der längsten Sequenz mit 0 gepolstert.
  • Zieldefinition:
    • Das letzte "Wort" (das letzte Element der Sequenz) wird als Ziel definiert.
  • Vorhersage:
    • Die letzten n = Sequenzlänge Wörter aus der Benutzereingabe werden in eine Sequenz für das Modell umgewandelt, kürzere Sequenzen werden gepolstert.
